# DSA Problem Solving in C++

A structured collection of Data Structures and Algorithms solutions in C++ from GeeksforGeeks and other coding platforms, with optimized approaches, complexity analysis, and problem notes.

## Overview
- Practice-focused repository for coding interviews and competitive programming.
- Each solution includes the problem link, core idea, and time/space complexity.

## Repository structure
- `Difficulty: Basic/` – fundamentals and warm-up problems.
- `Difficulty: Easy/` – standard interview questions.
- `Difficulty: Medium/` – pattern-based and optimization problems.
- `Difficulty: Hard/` – complex problems like “The Sequential-Set Closure Conundrum”.

## How to run
1. Clone the repo:
   ```bash
   git clone https://github.com/Techhyguy/DSA-Problem-Solving.git
   cd DSA-Problem-Solving
   ```
2. Navigate to any problem folder and compile:
   ```bash
   g++ problem_name.cpp -o problem
   ./problem
   ```

## Topics & patterns
This repository focuses on:
- Arrays, strings, linked lists, stacks & queues, trees, graphs, dynamic programming and more.
- Common patterns: two pointers, sliding window, recursion, greedy, divide & conquer, DP tabulation & memoization.
